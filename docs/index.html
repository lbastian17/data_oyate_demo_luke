Hello Makah Community Members!

We are working on creating an automatic speech recognition (ASR) model for Makah in order to assist in language preservation efforts! This is the technology behind voice assistants like Siri and educational apps like Duolingo. With enough audio data, similar features and other preservation tools can be created for Makah!!

See the About section to learn more about how your audio data is being used (and how it's being protected). Click below when you are ready to begin recording!
